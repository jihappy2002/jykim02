  3. 숙제

중요정보

  • 리스트 처리 태그
    1. ul태그 - Unordered list의 약자로 순서 없이 o 또는 ㅁ 표시로 리스트를 표기할 때 사용
    2. ol태그 - Ordered list의 약자로 앞에 숫자를 표기하여 순서대로 리스트를 표기할 때 사용
    3. li태그 - ul, ol 태그가 상위 태그 역할을 하고 li태그는 하위 태그 역할을 한다.
      따라서, li태그는 반드시 ol, ul과 같은 태그로 감싸져 있어야한다.
  • 링크 처리 태그
    1. a태그 - anchor의 약자로 필요한 정보를 링크화하여 연결할 때 사용
    2. 링크를 연결할 때는 a 태그에 이어 href(하이퍼텍스트 레퍼런스의 약자)를 이용한다.
      예시 - <*a href="oo">__<*/a>
    3. 링크를 새탭으로 띄우고자 할때는 target을 이용한다.
      예시 - <*a href="oo" target="_blank">__<*/a>
